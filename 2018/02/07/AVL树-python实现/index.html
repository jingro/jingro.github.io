AVL树-python实现

Feb 7, 2018

AVL树是平衡树的一种，在AVL树中插入、查找、删除操作和在 一棵普通的二叉查找树中一样，所不同的是在查找和删除之后要进行重新平衡

若定义结点平衡因子为结点的左子树高度减去右子树的高度:
bf=height(p.left)-height(p.right)

AVL树的平衡条件是任何一个节点的左子树和右子树的高度差都小于等于1,即所有结点的平衡因子的绝对值都小于1

对节点重新平衡所进行的操作的过程即四种旋转

  1. LL旋转
  2. LR旋转
  3. RR旋转
  4. RL旋转
  1. LL旋转即为：当前结点的平衡因子为2，结点的左子结点的平衡因子为1或0时所进行的操作，具体为对当前结点进行一次右旋；
  2. LR旋转即为：当前结点的平衡因子为2，结点的左子结点的平衡因子为-1时进行的操作，具体操作为对当前结点左子节点进行一次左旋，再对当前结点进行一次右旋；
  3. RR旋转即为：当前结点的平衡因子为-2，结点的右子结点的平衡因子为-1或0时进行的操作，具体操作为对当前结点进行一次左旋；
  4. RL旋转即为：当前结点的平衡因子为-2，结点的右子结点的平衡因子为1时进行的操作，具体操作为对当前结点的右子结点进行一次右旋，再对当前结点进行一次左旋；
